**Key insights**  
Developers should adopt `logit_mean`/`logit_sigma` immediately and update documentation/tooling. The deprecation path is well-handled with clear warnings and automatic conversion. Note that `logit_sigma` represents standard deviation (division), not variance, aligning with typical normalization semantics.

Additionally, the hash changes for events could affect any systems that rely on exact hash values for deduplication or comparison.

**Key insights**  
Developers must update any custom KV event subscribers to process the `group_idx` field for proper HMA support. The PR includes comprehensive tests validating group-aware event emission and hash behavior, which should be referenced when integrating with HMA-aware routing logic. Ensure that enabling KV events no longer inadvertently disables HMA features.
